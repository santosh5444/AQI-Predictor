### рҹ“Ӣ Prerequisites
- **Node.js** v18+ ([Install](https://nodejs.org))
- **Python** 3.10+ ([Install](https://python.org))
- **Git** ([Install](https://git-scm.com))

### рҹ’ҫ Clone & Setup

```bash
# 1пёҸвғЈ Clone repository
git clone https://github.com/yourusername/aqi-predictor.git
cd AQI-Predictor

# 2пёҸвғЈ Frontend Setup
npm install

# 3пёҸвғЈ Backend Setup
cd backend
npm install

# 4пёҸвғЈ Environment Variables

# - OpenWeather API
# - Mapbox Token
# - Database URL
```

### рҹ”Ҙ Run Development

```bash

# Terminal 2: Backend (http://localhost:3000)

# Terminal 3: ML Server (optional)
cd backend && python ml_server.py
```
